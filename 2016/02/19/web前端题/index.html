1.js有哪些数据类型呢？

答：[String, Number, Undefined, Boolean, Null, Object, Function, Array]

2.这些数据类型，哪些是引用类型的呢？

答：[Object, Function, Array]

3.说说函数表达式和函数声明的区别？

函数表达式：var func = function () {}

函数声明： function func () {}

两者的区别：当使用函数声明的形式来定义函数时，可以在声明之前被调用。函数表达式的形式来定义函数则不可以。会报错的。

4.闭包是什么？优缺点？

闭包的实质是一个函数，是一个用于返回局部变量值的函数，因为在全局中，受JavaScript链式作用域结构的影响，父级变量中无法访问到子级的变量值，为了解决这个问题，才使用闭包这个概念。由于闭包时，变量的值都保存到内存中，会导致页面加载时内存消耗很大，IE会导致内在泄露，因此尽量少用或用时要及时删除变量。

5.splice()的用法

index:起始位置；
howmany：要删除的长度；
item1,…,itemX:添加的项；
splice(index,howmany,item1,…,itemX)

var a =[1,2,3]; 
a.splice(1,1,2,3,4); 
console.log(a);//[1, 2, 3, 4, 3]
6.解释下浮动和它的工作原理？清除浮动的技巧？

浮动元素脱离文档流，不占据空间。浮动元素碰到包含它的边框或者浮动元素的边框停留。

  1. 使用空标签清除浮动。这种方法是在所有浮动标签后面添加一个空标签 定义css clear:both. 弊端就是增加了无意义标签。
  2. 使用overflow。给包含浮动元素的父标签添加css属性 overflow:auto; zoom:1; zoom:1用于兼容IE6。

7.详细解释ajax的工作原理？
  1. 创建ajax对象(XmlHttpRwquest/ActiveXObject(Microsoft.XMLHttp))
  2. 判断数据传输方式(get/post)
  3. 打开链接open()
  4. 发送send()

当ajax完成第四步(onreadystatechange)数据接收完成，判断http响应状态，如果为200-300之间或者304(缓存)，执行回掉函数

8.innerHTML，innerText和outerHTML，outerText的区别？

举个例子来说吧。

<div><span>内容</span></div>

使用这几个来获取上面div的内容的话，区别如下：

innerHTML:"<span>内容</span>",带有html标签

innerText:"内容",不带html标签

outerHTML:"<div><span>内容</span></div>"

outerText:获取元素跟innterText是一样的。
