apply-call-bind的用法以及他们的区别

发表于 2016-02-17   |  

一、call、apply用法

1.1、For Example
var one = {
      content : "结束吧！",
      year : "2015年是去年了。",
      print : function() {
        console.log(this.content + this.year );
      }
}

one.print();//结束吧！2015年是去年了。

这样one.print()的打印结果是显而易见的不用多说。

var two = {
      content : "时间很快~",
      year : "今年是2016年了。",
}

现在我们新增加一个 two对象。那么需求来了：two对象本身没有print方法，那么如何用one对象里面的print方法来显示数据呢？那就可以用到call或者apply了。

one.print.call(two);  //时间很快~今年是2016年了。
one.print.apply(two); //时间很快~今年是2016年了。
1.2 应用场景

1.2.1 数组之间的追加

var arr1 = [1 , 2 , 3 , 4]; 
var arr2 = [5 , 6 , "name"]; 
Array.prototype.push.apply(arr1, arr2); 
console.log(arr1);//[1, 2, 3, 4, 5, 6, "name"]

1.2.2 获取数组中的最大值和最小值

var arr3 = [5, 4 , 120 , -230 ]; 
var maxValue = Math.max.apply(Math, arr3),   //120
    maxValue = Math.max.call(Math,5, 4 , 120 , -230); //230

数组本身没有max方法。但是Math 有，我们就可以借助 call 或者 apply 使用其方法。

二 、call和apply之间的区别

说到这两者之间的区别，就得先看看他们的定义了。你就不难发现其中的区别。

2.1 call方法 

调用一个对象的一个方法，以另一个对象替换当前对象。 

call([thisObj[,arg1[, arg2[, [,.argN]]]]]) 

参数：

thisObj:可选项。将被用作当前对象的对象。 

arg1, arg2, , argN:可选项。将被传递方法参数序列。

2.2 apply方法 

应用某一对象的一个方法，用另一个对象替换当前对象。

apply([thisObj[,argArray]]) 

参数： 

thisObj:可选项。将被用作当前对象的对象。

argArray:可选项。将被传递给该函数的参数数组。

2.3 For Example 

var func = function(arg1, arg2) {};
func.call(this, arg1, arg2);
func.apply(this, [arg1, arg2]);

其中 this 是你想指定的上下文，他可以是任何一个 JavaScript 对象(JavaScript 中一切皆对象)，call 需要把参数按顺序传递进去，而 apply 则是把参数放在数组里。

可以看出两者的区别：对于 apply、call 二者而言，作用完全一样，只是接受参数的方式不太一样。

三、bind的用法

For Example

var func= function(){
    console.log(this.name);
}
var foo = {
    "name" : "whyFind"
}
var usebind = func.bind(foo);
usebind(); //whyFind

MDN的解释是：bind()方法会创建一个新函数，称为绑定函数，当调用这个绑定函数时，绑定函数会以创建它时传入 bind()方法的第一个参数作为 this，传入 bind() 方法的第二个以及以后的参数加上绑定函数运行时本身的参数按照顺序作为原函数的参数来调用原函数。

在一道非常经典的面试题中就可以用bind来解决实际问题：

var foo = function(){
    this.myName = "Hello, World!";
}
foo.prototype.sayHello = function(){
    console.log(this.myName);
}
foo.prototype.bar = function(){
    setTimeout(this.sayHello, 1000);
}
var f = new foo;
f.bar(); //undefined

这里的myNmae并不能被正确打印出来。那么怎么办呢？看这里。。

var foo = function(){
    this.myName = "Hello, World!";
}
foo.prototype.sayHello = function(){
    console.log(this.myName);
}
foo.prototype.bar = function(){
    setTimeout(this.sayHello.bind(this), 1000);
}
var f = new foo;
f.bar(); //Hello, World!

这样就能正确的打印出来了。

四、总结

1.apply 、 call 用法功能大致相同，只是传入的参数的方法不一样；

2.apply 、 call 、bind 三者都是用来改变函数的this对象的指向的；

3.apply 、 call 、bind 三者第一个参数都是this要指向的对象，也就是想指定的上下文；

4.apply 、call 则是直接返回结果(立即调用)；bind 是返回对应函数体，稍后应该再调用一次。
